Boxes

As with every provider, the Parallels provider has a custom box format.

The easiest way to use a box is to add a box from the Atlas website. You can also add and share your own customized boxes there. Read more on the Atlas Help page.

Discovering boxes

The easiest way to find boxes is to look on the public Vagrant box catalog for a box matching your use case: all boxes with the Parallels provider support

Official boxes compatible with Parallels provider are available on the parallels account page

Adding a box from the catalog is very easy:

$ vagrant box add parallels/ubuntu-14.04
...

You can also quickly initialize a Vagrant environment with command

$ vagrant init parallels/ubuntu-14.04
...
